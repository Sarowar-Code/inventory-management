# Inventory Management - POS Dashboard

This is a scaffolded Next.js 15 (app router) project with Tailwind CSS and a desktop-first POS and Inventory management UI.

Features:
- Desktop-first fixed sidebar and header layout
- POS page with product grid and billing/cart sidebar
- Inventory table with low stock badges
- Dashboard with top metric cards
- Deep blue and slate grey color palette

Setup (Windows PowerShell):

```powershell
# install
npm install

# dev
npm run dev
```

Notes:
- Images in /public/images are placeholders (empty files). Replace them with product images.
- Tailwind and shadcn components are approximated; adjust per project needs.
